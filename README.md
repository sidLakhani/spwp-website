# Sincerely, People With Periods

## Author
**Sid Lakhani**

## About
Website for Sincerely, People With Periods (SPWP) — a nonprofit organization dedicated to ending period poverty.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```
